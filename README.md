因为这个插件有段时间，无法登录使用了。所以我把登录使用去掉了。

通信鉴权，秘钥默认为666666

所以服务端的`authorization.txt`文件，内容写成666666就可以了。

此外你担心鉴权秘钥被一些人撞到了...你完全可以把处理接口url写的复杂点or无厘头(xyxnotice123)，都可以解决问题的。
